SVG CON CSS

Maximiliano Ramírez Olea & Emmet Chavez Flood

Esta página indica 3 formas de implementar En esta página les mostraremos distintas maneras de aplicar CSS (Cascading Style Sheets) con svg en torno a la temática de "celular".

Una animación con CSS

Esta animación generada gracias a la propiedad de "transform" representa la acción de cargar el celular usando "transalate", y la respuesta de éste al aparecer un rayo amarillo en el que se uso "opacity".

Una transición con CSS

Esta transición generada gracias a la propiedad de "transition" representa el nivel de carga, mediante la volarización por los colores y la posición que le condice a cada rectangulo con respecto al total, usando "opacity", "transition" y "cursor".

Una transición con CSS y algo más

Esta transición representa el rol del celular en la comunicación dentro de un mundo globalizado, es por eso que se elige el fondo de la aplicación de mensajería WhatsApp; fondo que se encuentra vinculado a un archivo .jpg online (Wallpaper WhatsApp) por lo que no forma parte del .svg original, el que solo proporciona la silueta. Se usó "opacity", "transition" y "fill".
